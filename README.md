# Sahariar Dev Portfolio

> A high-performance, statically generated developer portfolio and digital resume, built to showcase enterprise-grade frontend engineering and AI integrations.

## System Architecture

This portfolio is not just a static site; it's a demonstration of modern web capabilities. It leverages Next.js React Server Components to minimize client-side JavaScript, ensuring lightning-fast load times and perfect Core Web Vitals.

### Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Framer Motion (for smooth, hardware-accelerated animations)
- **Deployment:** Vercel Edge Network
- **Analytics:** Vercel Web Analytics

## Core Engineering Highlights

- **Server-Side Rendering (SSR) Optimization:** Aggressively caches static assets and utilizes React Server Components to deliver content instantly to the client.
- **Fluid Micro-Interactions:** Custom Framer Motion variants create a premium, glassmorphic UX without sacrificing rendering performance.
- **Responsive Architecture:** A fully fluid layout that dynamically adapts from ultrawide monitors down to mobile viewports using pure CSS Grid and Flexbox techniques.
- **Semantic HTML & SEO:** Strictly adheres to accessibility (a11y) standards and structural semantic HTML to ensure optimal SEO indexing and screen reader compatibility.

## Setup & Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Build for production: `npm run build`
